PT Tools

Quick clinical calculators and note helpers

Tools Info

Info

PT Tools is a simple collection of clinical calculators and note helpers built for fast use on a phone or computer. The goal is to make common rehab calculations quicker, easier to interpret, and easier to copy into documentation.

This tool is intended as a clinical support resource and not as a replacement for clinical judgment. Do not enter patient-identifying information.

Current Version: 0.1.0
About

Created by Daniel Ballou, PT, DPT, OCS, COMT. This project is being tested and expanded based on real-world clinical use and feedback.

Roadmap
  • Add more calculators for strength, balance, outcomes, and exercise prescription.
  • Improve note-ready output and patient-friendly explanations.
  • Add favorites, recent tools, and faster mobile workflows.
  • Add export/share options and cleaner report formatting.
  • Refine thresholds, interpretation language, and overall UX based on testing.
Changelog
  • 0.1.0 — Initial public testing version with BMI, gait speed, heart rate zones, utilities, strength, balance, and outcomes tools.
  • Added structured output with copy buttons for summary, explanation, note, raw values, and full result.
